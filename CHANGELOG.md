# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [0.1.4] - 2026-01-22

- Update .gitignore to retain package-lock.json for CI/CD reproducibility; add package-lock.json file for dependency management.


## [0.1.3] - 2026-01-22

- Update stable tag in README.md to version 0.1.2
- Update sync-version script to modify README.md stable tag during version sync


## [0.1.2] - 2026-01-22

- Delete navigation-taxonomies-plugin-prompt.md
- Enhance taxonomy navigation and selection functionality. Improved event handling for taxonomy select buttons with multiple strategies to locate associated select elements. Updated taxonomy navigation block to support submenus and ensure fresh data retrieval, including cache invalidation on term updates. Added logging for better debugging and tracking of navigation block rendering.
- Sure! Pl


## [0.1.1] - 2026-01-22

- Remove the navigation-taxonomies-plugin-prompt.md file, which contained the initial plugin requirements and specifications for a WordPress plugin that auto-syncs taxonomy navigation. This deletion streamlines the project by eliminating outdated documentation.
- Enhance taxonomy navigation and selection functionality. Improved event handling for taxonomy select buttons with multiple strategies to locate associated select elements. Updated taxonomy navigation block to support submenus and ensure fresh data retrieval, including cache invalidation on term updates. Added logging for better debugging and tracking of navigation block rendering.
- Sure! Pl


## [0.1.0] - 2026-01-22

### Added
- Initial release of WE Taxonomy Navigation
- Taxonomy Navigation block with full navigation styling support
- Taxonomy List block with multiple display options (dropdown, filterable, list)
- Automatic term update functionality
- Hierarchical submenu support
- Template part support
- Cache management for fresh content
- GitHub update system integration
